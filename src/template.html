Hello, Webpack!

This is a development environment that uses Webpack to generate html, bundle preprocessed scripts / styles, and watch them with Webpack Dev Server hot reload.

This also contains a production environment that minifies JS and uses Extract Text Webpack Plugin to generate minified CSS.

Other core tools included are Sass, Autoprefixer, Stylelint, and ESLint.

Built With

  • Webpack Dev Server - Local Node Server for Webpack

  • Sass - CSS with superpowers
  • Autoprefixer - adds vendor prefixes to CSS rules
